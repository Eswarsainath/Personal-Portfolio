I have created Personal Portfolio website using HTML, CSS and BootStrap.
I have included various bootstrap classes. All the included classes are listed below.

Container - Containers are used to contain, pad, and (sometimes) center the content within them
Row - Rows are wrappers for columns
Gutters - Gutters are the padding between your columns, used to responsively space and align content in the Bootstrap grid system.
flex - Used to manage the layout and alignment
progress bar - Used to indicate the progress
card - flexible and extensible content container
collapse - Toggle the visibility of content
form - Used for contact form
grid -  uses a series of containers, rows, and columns to layout and align content
Input - Used to take input from the form
button - Used button 
Icon - used various font awesome icons
